type TbitCrushParam = record

  sourceDepth: Byte;

  

The current Depth of the PCM in bits (ex. 16 for wave PCM)

  crushDepth: Byte;

  

What you want the PCM to be crushed to in bits (ex 12, 14, 8 bits)

end;

Description

Two values are needed. 1. crushDepth - what you want the PCM to be crushed to in bits (ex 12, 14, 8 bits) 2. sourceDepth - The current Depth of the PCM in bits (ex. 16 for wave PCM)
